# univer-api-2

```
npm i
```
#### для старта в режиме разработки

```
npm run dev
```
#### для старта в режиме тестирования с модулем mocha
```
npm run test
```
#### запуск форматирования eslint
```
npm run lint
```
#### запуск форматирования prettier
```
npm run format
```
